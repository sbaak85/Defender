# EMBERWATCH — 地城守望

縱向正面俯視的 6×9 棋盤格塔防遊戲。所有遊戲檔案與圖片素材都在本資料夾內。

## 開始遊戲

可直接雙擊 `index.html` 開啟，或雙擊 `Start-Game.cmd` 透過本機 HTTP 服務執行。

## 操作

- 點擊標題畫面的 `Tap to continue`，經過 3、2、1 後開始。
- 從底部單位卡拖曳到塔樓走廊，或先點卡片再點空位。
- 點擊已部署單位後可按「回收 50%」。
- 「維修 +50」消耗 100 金幣。
- 右上可切換 ×1、×5、×20，方便快速測試三分鐘關卡。
- 空白鍵暫停／繼續。

## 完整機制

- 180 秒逐秒倒數；城牆血量 500，時間歸零且城牆仍在即勝利。
- 走廊開場即開放全部 6 格；已部署守軍可免費拖曳到其他空格，拖到另一名守軍上會直接互換位置，也可依序點選兩名守軍完成交換。
- 5 種敵軍由弱至強解鎖；每次精準移動一格，整體移動速率已降低 30%，目前約 1.43～4.29 秒一格。
- 5 種守軍依指定射速、射程、傷害與範圍自動攻擊。
- 完整金幣派遣、擊殺獎勵、50% 回收與城牆維修。
- 敵人重疊會左右錯位；受傷白閃、浮動傷害、血條扣減、倒下淡出。
- 開場、部署、攻擊、砲擊、受傷、金幣、維修、勝敗均有即時合成音效。
- 十種 AI 生成角色／塔台素材採約 3 頭身美式動畫比例；我方背面朝上、敵方正面朝下。原始洋紅幕圖保留於 `assets/source/`，遊戲直接載入 `assets/processed/` 內已完成去背的透明 PNG。
- 戰場單位、已部署守軍與底部五個單位選項皆使用透明 PNG；角色圖片約放大為原尺寸的 240%，圖片以完整包含方式呈現，不裁切頭部、腳或武器。放大僅影響視覺，單位定位與互動判定仍維持一格。

素材清單詳見 `ASSETS.md`；完整規則與開發交接詳見 `HANDOFF.md`。
